Почта для бизнеса Microsoft Exchange

Корпоративная почта для сотрудников в облаке — это сервис, объединяющий возможности электронной почты Outlook, календаря и менеджера задач. Вы получаете готовую к работе систему, которая легко масштабируется и настраивается. Дополнительно можно подключить услугу облачного хранилища NextCloud для обмена файлами через почту.

Решение Microsoft Exchange остаётся доступным, несмотря на уход Microsoft с российского рынка. В зависимости от ваших задач, можно арендовать несколько почтовых ящиков для сотрудников (модель SaaS) или облачный почтовый сервер для организации целиком (модель IaaS).

  • Удобная рабочая среда Синхронизируйте почту, задачи, календари и контакты с ПК и мобильными девайсами.
  • Антиспам и антивирус Вы получаете автоматическую защиту от спама и вирусов для вашей почты.

Синхронизация почты с AD

Возможна синхронизация с вашей AD, автоматическое добавление пользователей, изменение, блокировка, синхронизация пароля (опционально). Наш модуль, который размещается на вашем сервере AD или любой машине в домене, с заданной периодичностью собирает данные и синхронизирует их с нашим Exchange.

Доступны два варианта настройки, с передачей хеша пароля (пользователи смогут входить в почту с тем паролем, что и в Active Directory) и без передачи хеша пароля, тогда пароль для почты потребуется изменить отдельно.

Часто задаваемые вопросы?

Как я могу использовать облачную почту для бизнеса?

Cоздать виртуальную машину (VM) можно в панели управления Cloud Director из имеющегося преконфигурируемого шаблона или из ISO образа. Для вашего удобства в консоли Cloud Director вы найдёте шаблоны всех распространённых OS: Ubuntu, CentOS, Debian, Windows Server и другие. Пошаговую инструкцию для создания виртуальной машины из шаблона или из ISO образа, и другие полезные инструкции вы сможете найти в базе знаний. Создать виртуальный сервер в облаке можно, обратившись в отдел продаж Cloud4Y. Решения для клиентов предоставляется на базе иностранного и отечественного ПО, включая решения импортозамещения. Облачный сервер можно развернуть на базе Windows и Linux. Если вы хотите создать облачное хранилище, можете воспользоваться этим руководством.

Для чего нужна корпоративная электронная почта?

Облачный сервер (или виртуальный сервер) — это виртуальная инфраструктура, которая позволяет пользователям хранить данные, выполнять вычисления и запускать приложения на удалённых серверах, которые находятся в дата-центрах. Вместо того, чтобы иметь физический сервер, который расположен на месте, можно использовать предоставляемые провайдером облачных услуг облачные серверы для работы или игр. Обращаем ваше внимание, что облачный сервер Cloud4Y для игр не используется.
